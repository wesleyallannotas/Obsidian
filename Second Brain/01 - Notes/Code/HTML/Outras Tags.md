# Introdução
Existem diversas tags ou melhor elementos HTML com seus usos específicos, e aqui os traremos de forma simplificada.

| Elemento | Função                              |
| -------- | ----------------------------------- |
| `<time>` | Tempo em horas `<time>15:00</time>` |
| `<sup>`  | Número elevado `10<sup>2</sup>`     |
